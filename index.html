Tags


table

É utilizada para definir uma tabela, porém ela não cria a tabela completamente, é necessário utilizar as tags "tr" e "dr" para criar as linhas e colunas.

Sintaxe:<table> </table >


del

É utilizada para definir um texto que foi excluído

Cite e Datetime: Cite define a fonte do motivo da exclusão do texto e Datetime define em que momento essa exclusão foi realizada.

Sintaxe:<del> </del>

Exemplo:Tag Del


ins

É utilizada para definir um texto que foi inserido ou alterado.

Cite e Datetime: Cite define a fonte do motivo da mudança do texto e Datetime define em que momento essa mudança foi realizada.

Sintaxe:<ins> </ins>

Exemplo:Tag Ins
